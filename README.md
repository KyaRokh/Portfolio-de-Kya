# Portfolio

Bienvenue sur la présentation de mon portfolio ! 

J'ai conçu ce projet dans le cadre de ma reconversion professionnelle en tant que Développeuse Web et Web Mobile (DWWM).

## Lien du projet
Découvrir le site en direct : [Mon Portfolio en Ligne](https://kyarokh.github.io/Portfolio-de-Kya/)

## Inspiration
Le design s'inspire du style graphique **Néo-Brutaliste** (bordures épaisses, contrastes marqués, ombres nettes) appliqué à un concept de classeur administratif à intercalaires, qui fait un rappel à mon parcours de 15 années d'expérience dans le secteur de l'administration et du médico-social.

## Technologies & Pratiques utilisées
* **HTML5 sémantique** : Structure optimisée pour le SEO et le flux de lecture.
* **CSS3 Moderne** : Design Responsive pour s'adapter aux écrans.
* **JavaScript (Vanilla)** : Gestion dynamique de la navigation par onglets sans framework.
* **Accessibilité (A11y)** : Implémentation des patterns du W3C.
